# Student Management System

## 🚀 Overview
The Student Management System is a fullstack web application designed to manage students, teachers, classes, and attendance efficiently.

This system is built to simulate a real-world academic environment, featuring role-based access control and structured workflows for managing class sessions and attendance tracking.

## 🔥 Features

### 👤 Role-Based Access Control

#### 🛠 Admin
- Manage teachers and students  
- Create and manage classes  
- Assign teachers and students to classes  
- Track attendance records  
- Full system control  

---

#### 👨‍🏫 Teacher
- View assigned classes  
- Start and end class sessions  
- Generate attendance sessions (scan or code input)  
- Automatically record student attendance  
- Review attendance records  
- Approve or reject student permission requests  

---

#### 🎓 Student
- View personal attendance and academic records  
- View classmates information  
- Join attendance sessions (scan or input code)  
- Submit permission requests (absence reasons)  

---

### 📊 Attendance Management System
- Session-based attendance (start / end class)  
- Real-time attendance tracking  
- Automatic absence marking after session ends  

---

### 🔐 Authentication & Security
- Role-based login system  
- Protected routes and secure access control  

---

### 📈 Dashboard
- Overview of system data  
- Track attendance and user activity  

---

## 🛠 Tech Stack

- Frontend: React.js  
- Backend: Laravel (REST API)  
- Database: MySQL  
- Styling: Tailwind CSS
